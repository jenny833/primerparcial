BIENVENIDOS A LA CARRERA DE INFORMATICA

La carrera de Informática con Orientación en Robótica es una de las nuevas carreras implementadas por la Secretaria de Educación de Honduras desde el año 2017. Esta carrera consiste en que los alumnos obtenga conocimientos que normalmente no entrarían en la malla de estudio de una carrera normal. En Honduras actualmente solo 21 Institutos poseen esta carrera,Incluyendo el Instituto Tecnico Francisco Miranda ya que para impartir las clases que esta conlleva se necesitan los recursos necesarios, los cuales en ocasiones no son de tan bajo costo. Los proyectos que realizan los alumnos de la carrera de Informática con Orientación en Robótica son muy variados llegando a hacer diferentes funciones y cada una con implementación para la vida diaria. Los estudiantes de Informática con orientación a Robótica desarrollan mayores capacidades de razonamiento para resolver problemas complejos. Si esta interesado en estudiar una carrera y te llama la atención esta, solo debes abocarte a algunos de los Institutos que cuentan con ella, así como realizar el proceso de admisión para poder ser aceptado como un alumno que cursara este bachiller.

Clases desarrolladas en Informatica
-Diseño Web -Mantenimiento y Reparacion - Programcion II -Producciones Digitales -Rredes Informaticas -Geston Empresarial Orientado Conceptos Clave en Informática: Hardware: Son los componentes físicos de una computadora, como la unidad central de procesamiento (CPU), la memoria RAM, el disco duro, la tarjeta gráfica, entre otros. Software: Es el conjunto de programas y aplicaciones que le dan instrucciones a la computadora para realizar tareas específicas. El software se divide en: Software de sistema: El sistema operativo (como Windows, macOS, Linux) y herramientas básicas que permiten que la computadora funcione. Software de aplicación: Programas que realizan tareas específicas para el usuario, como procesadores de texto, hojas de cálculo, navegadores web, etc. Software de desarrollo: Herramientas y lenguajes de programación para crear nuevos programas (por ejemplo, Python, Java, C++). Redes y comunicaciones: La informática también está involucrada en la transmisión de datos a través de redes, como Internet. Esto incluye protocolos de comunicación, redes locales (LAN), redes amplias (WAN), y tecnologías de comunicación como Wi-Fi, 5G, etc. Bases de datos: La informática permite el almacenamiento y recuperación eficiente de grandes cantidades de datos mediante el uso de bases de datos, que se gestionan con sistemas de gestión de bases de datos (DBMS) como MySQL, PostgreSQL, o SQL Server. Algoritmos: Son secuencias de pasos o instrucciones definidas para resolver un problema específico. Los algoritmos son esenciales en la programación y en la toma de decisiones dentro de los sistemas informáticos. Inteligencia Artificial (IA): Un campo de la informática dedicado a la creación de sistemas que pueden simular procesos de inteligencia humana, como el aprendizaje automático (machine learning), el procesamiento de lenguaje natural, la visión computacional, y la automatización de tareas.
